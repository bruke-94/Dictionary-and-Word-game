# Dictionary & Word Game Website

A simple website built using **HTML, CSS, and JavaScript**.

## Project Description

The website has two main features:

- A **Dictionary page** where users can search for word meanings
- A **Word Guess Game** where users guess a hidden word letter by letter

## Features

- Search for word definitions using a public dictionary API
- word guessing game
- Game progress is saved using **localStorage**

## Technologies Used

- HTML5
- CSS3
- JavaScript
